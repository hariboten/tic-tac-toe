# エージェントタブUI整理 実装計画

## 目的
エージェントタブの操作を「学習」「データ管理」「比較」の3導線に分離し、目的ごとに迷わず操作できるUIへ改善する。

## 対象範囲
- `src/app/app.component.html`
- `src/app/app.component.ts`
- `src/app/app.component.scss`
- `src/app/app.component.spec.ts`

## 実装方針
1. エージェントタブ内にサブタブを追加し、表示領域を切り替える。
2. 画面上部に状態サマリーを追加する（対象プロファイル / 学習状態 / 最新メッセージ）。
3. 危険操作（学習データクリア、保存データ削除）は確認ダイアログを挟む。
4. 比較テーブルは選択中プロファイルを強調表示し、並び替え基準を持たせる。

## 受け入れ観点
- 既存の学習・保存・読み込み・JSON入出力が引き続き動作する。
- サブタブ切り替えで必要な要素だけが表示される。
- 危険操作は確認なしで実行されない。
- 比較表示で選択中プロファイルが判別できる。

## テスト方針
- 既存の `app.component.spec.ts` を更新し、以下を確認する。
  - サブタブ初期表示
  - データ管理タブからJSONエクスポート/インポート可能
  - 比較タブで選択中プロファイル強調
  - 危険操作の確認ダイアログが呼ばれる
